Cooperatives Stats

This is the Frontend website for CoStats.
This project provides an interface for both the cooperative manager to submit relevant data to feed the database, as well as an interface for national admins to keep track of the performance and strength of said cooperatives.

This is a prototype Backend server of a much larger project made for Cooperation Development Office (ODCO: A Government institute to help the development of local cooperatives) in Morocco. This backend provide a database that consist of all the registered Cooperatives, and for each its Field, Members, Profits and Losses, as well as other indicative of a cooperative performance such as its yearly events, meetings, advertisements, etc.

BackEnd : Spring Boot Google Cloud App Engine Cloud Build SQL Firebase Firestore DevOps
FrontEnd: Node.JS React Axios Firebase Web Hosting
